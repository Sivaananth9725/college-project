# Crop Disease Management System

A full-stack application for detecting and managing crop diseases using AI.

## Features

- Disease detection using machine learning
- Weather integration
- Chatbot for farmer assistance
- Admin panel
- Interactive map

## Backend (Django)

- API for disease detection
- Weather service
- Groq chatbot integration
- Alert system

## Frontend (React)

- Farmer dashboard
- Disease detection interface
- Chatbot
- Map visualization

## Setup

### Backend

1. cd backend
2. pip install -r requirements.txt
3. python manage.py migrate
4. python manage.py runserver

### Frontend

1. cd frontend
2. npm install
3. npm start

## Deployment

- Backend: Render
- Frontend: Vercel
